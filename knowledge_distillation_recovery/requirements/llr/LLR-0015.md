## Description

`kdr.quant.native_backend` provides STE simulators for INT3, INT2 (uniform symmetric integer quant) and any KV format modelopt's installed version doesn't ship (queried via `feature_matrix.py`).

## Rationale

Fills holes in modelopt's coverage so the YAML's recipe space is complete.

## Acceptance Criteria

- [ ] Hypothesis property tests on INT3 and INT2 simulators: identity-on-forward, dequant-quant idempotence, gradient flow through STE
- [ ] Granularity correctness: K per-channel along head_dim, V per-token along seq_len
