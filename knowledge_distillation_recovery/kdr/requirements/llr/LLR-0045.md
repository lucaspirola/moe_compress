## Description

`tests/property/` directory contains hypothesis property tests for every NativeBackend simulator function, covering identity-on-forward, dequant-quant idempotence, granularity correctness, and gradient flow.

## Rationale

Property tests catch axis/sign/scale bugs that example-based tests miss.

## Acceptance Criteria

- [ ] At least one `@hypothesis.given` test per public NativeBackend function
- [ ] Test failures produce minimised counter-examples
