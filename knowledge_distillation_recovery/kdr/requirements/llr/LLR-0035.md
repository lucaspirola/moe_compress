## Description

Bootstrap installs Zyphra's transformers fork via `pip install --upgrade --force-reinstall "transformers @ git+https://github.com/Zyphra/transformers.git@zaya1"` BEFORE invoking the trainer.

## Rationale

Required for ZAYA1's `ZayaForCausalLM`; harmless on other models since the fork tracks upstream.

## Acceptance Criteria

- [ ] Install happens after `git clone` of moe_compress and before `python -m kdr.train`
- [ ] Install failure is fatal (exit 3 with a clear error)
