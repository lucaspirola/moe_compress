## Description

`docker/bootstrap.sh` validates `HF_TOKEN`, `STUDENT_REPO`, and `CACHE_MOUNT` env vars at the top; missing → exit 2 with a clear error.

## Rationale

Fail fast before downloading 17 GB of teacher weights.

## Acceptance Criteria

- [ ] Script is `set -euo pipefail` from line 1
- [ ] Each missing env var produces a distinct error message naming the var
