## Description

After `_load_teacher` constructs the teacher model, call `teacher.requires_grad_(False)` and assert no parameter has `requires_grad=True` before returning the model.

## Rationale

Gradient flow into the teacher would couple it to the student and break the soft-target contract.

## Acceptance Criteria

- [ ] Post-condition assertion runs in non-release mode
- [ ] A unit test runs a backward pass and asserts every teacher param has `grad is None` afterwards
