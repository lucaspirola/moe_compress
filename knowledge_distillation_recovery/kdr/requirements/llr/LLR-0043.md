## Description

FKLD distillation iterates the full calibration tensor up to `cfg.distillation.total_tokens`, sourcing token batches from the same packed tensor used by PTQ calibration.

## Rationale

Eliminates calibration-vs-training data drift.

## Acceptance Criteria

- [ ] Token counter advances over the full tensor
- [ ] When `total_tokens > calibration.num_sequences * sequence_length`, the loop wraps via re-shuffle (with logging)
