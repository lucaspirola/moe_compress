## Description

`kdr.training.zero3_init.is_zero3(accelerator) -> bool` returns True iff the accelerator is configured with DeepSpeed ZeRO stage 3. This helper is consumed by BOTH HLR-0010 (optimizer dispatch — DSCPUAdam vs bnb-8bit) AND HLR-0009 (eval — `GatheredParameters` wrapper for `.generate`); it is parented under HLR-0010 only by file convention but is shared infrastructure across both HLRs. See LLR-0048 for the related `activate_zero3_init` context manager.

## Rationale

Used to gate the `_activate_zero3_init` and `GatheredParameters` paths.

## Acceptance Criteria

- [ ] Returns False outside DeepSpeed
- [ ] Returns True only when stage == 3 (not stage 1 or 2)
- [ ] Cross-HLR consumption is documented in this LLR's Description so `req trace` users see both consumer HLRs
