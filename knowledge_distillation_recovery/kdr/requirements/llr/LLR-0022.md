## Description

`kdr.adapters.base.ModelAdapter` is a `typing.Protocol` declaring `load_teacher_and_student`, `attention_module_paths`, `kv_quant_exempt_indices`, `fp32_carve_outs`, `required_attn_implementation`, `router_replay_hook`.

## Rationale

Single abstraction for all model-family knowledge.
