## Description

Define `kdr.quant.specs.Format = Literal['int','fp8','mxfp4','nvfp4']` and `Granularity = Literal['tensor','channel','group','block','token']` as the canonical enums used everywhere.

## Rationale

Single source of truth for valid format/granularity values; pattern matches against this enum can be checked exhaustively by mypy.

## Acceptance Criteria

- [ ] These types are imported by both KVQuantSpec and WeightQuantSpec
- [ ] ModelOptBackend's config_map exhaustively maps every format value
