## Description

When `teacher.config.torch_dtype == 'float8_e4m3fn'` (or an equivalent FP8 dtype), explicitly cast the teacher's `lm_head` linear layer back to BF16 before freezing.

## Rationale

FP8 lm_head produces unstable softmax outputs at the vocab tail; BF16 lm_head with FP8 body weights is the documented stable configuration on H200.

## Acceptance Criteria

- [ ] Code path is exercised only for FP8 teachers
- [ ] Post-cast assertion verifies `teacher.lm_head.weight.dtype == torch.bfloat16`
