## Description

ZAYA1 adapter's `fp32_carve_outs` returns module path patterns matching `lm_head`, the routing softmax modules, CCA cache state, RMSNorm modules, and residual-stream addition points.

## Rationale

Mirrors arxiv:2605.05365 §IV-D; failing to carve these out destabilises QAD by promoting numerical precision exactly where ZAYA1 is most sensitive.

## Acceptance Criteria

- [ ] List literally includes `lm_head`
- [ ] List patterns are tested against an instantiated ZAYA1 model and resolve to non-empty submodule sets
