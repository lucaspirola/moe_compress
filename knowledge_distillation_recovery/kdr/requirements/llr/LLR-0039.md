## Description

`kdr.training.optim.build_optimizer(student, dconf)` switches on `dconf.optimizer`: `'adamw_bnb_8bit'` → `bitsandbytes.optim.AdamW8bit`; `'deepspeed_cpu_adam'` → `DeepSpeedCPUAdam(adamw_mode=True)`.

## Rationale

Direct port from `structural_recovery/distillation.py:159-196`.

## Acceptance Criteria

- [ ] Both branches return optimizer instances configured with `learning_rate`, `betas`, `weight_decay` from `dconf`
- [ ] Unknown optimizer values raise ValueError naming the unknown value and listing valid ones
