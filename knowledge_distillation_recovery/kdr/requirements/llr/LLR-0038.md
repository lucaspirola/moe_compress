## Description

All `log.info` and `print` calls in `kdr.eval.quick` are guarded by `if accelerator.is_main_process:`.

## Rationale

Avoids interleaved log spam from N ranks.

## Acceptance Criteria

- [ ] No log message escapes the rank-0 guard
- [ ] PPL value is logged exactly once per eval call
