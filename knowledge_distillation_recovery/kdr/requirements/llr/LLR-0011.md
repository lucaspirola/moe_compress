## Description

`kdr.config.QuantBlock` aggregates `weight: WeightQuantSpec`, `kv_quant: KVQuantBlock` (where KVQuantBlock has fields `key: KVQuantSpec` and `value: KVQuantSpec`).

## Rationale

Mirrors the YAML's nested shape so loading is direct.

## Acceptance Criteria

- [ ] YAML with `quant: {weight: ..., kv_quant: {key: ..., value: ...}}` deserialises cleanly
- [ ] Missing `kv_quant.key` or `kv_quant.value` raises ValidationError
