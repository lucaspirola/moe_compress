## Description

In `da_qad` mode, modelopt's `calibrate_forward_loop` consumes exactly `cfg.calibration.ptq_subset_size` sequences from the front of the calibration tensor produced by `_tokenize_to_fixed_length`.

## Rationale

First-N split avoids drift between calibration and training distributions.

## Acceptance Criteria

- [ ] Subset is contiguous from index 0
- [ ] Test asserts modelopt receives ≤ ptq_subset_size batches
